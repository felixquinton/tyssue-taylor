# tyssue-taylor - From experimental data to a tyssue model

This package uses the [`tyssue`](https://tyssue.readthedocs.io) library to adjust a vertex model to experimental microscopy data.

## Dependencies

- python >= 3.6
- tyssue >= 0.2
- python-opencv
